# SubastaMaximo

Demo: https://subasta-maximo.vercel.app

Descripción
Proyecto de frontend realizado con React + Vite. Simula una plataforma simple de subastas donde los usuarios pueden ver ítems y hacer pujas (actividad de práctica / proyecto personal).

Tecnologías
- React
- Vite
- JavaScript (ES6+)
- HTML / CSS

Cómo ejecutar (local)
1. `git clone https://github.com/Mateof311/SubastaMaximo.git`
2. `cd SubastaMaximo`
3. `npm install`
4. `npm run dev`
5. Abrir `http://localhost:5173` (o el puerto que indique Vite)
